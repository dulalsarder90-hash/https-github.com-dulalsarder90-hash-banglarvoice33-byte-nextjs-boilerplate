# Banglar Voice

Banglar Voice is an AI-Powered Bengali News, Media and Digital Journalism Platform.

## Mission
Deliver credible, independent and technology-driven journalism for Bengali-speaking audiences.

## Vision
Build a sustainable Bengali media ecosystem powered by innovation, data and community engagement.

## Features
- Bengali News Portal
- AI Assisted Journalism
- Breaking News
- Multimedia Content
- Investigative Reporting
- SEO Optimization
- Responsive Design

## Technology Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Docker
